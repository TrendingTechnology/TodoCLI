# TodoCLI

A Swift command-line Todo App to manage tasks.

TodoCLI helps you with:
 - [ ] Save a todo
 - [ ] List all todos
 - [ ] Mark a todo as completed
 - [ ] Delete a specied todo
 
### Requirements
 - XCode 11.4 beta 1 and up
